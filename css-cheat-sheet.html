CSS Cheat Sheet

Basic Syntax

selector { property: value; }: CSS rule structure, where a selector targets an HTML element, and properties define the styles applied to it.

Selectors

*: Universal selector, which applies styles to all elements on the page.

element: Tag selector, which targets specific HTML elements by their tag name (e.g., p, h1).

.class: Class selector, which targets elements with a specific class attribute (e.g., .my-class).

#id: ID selector, which targets a specific element with a unique ID attribute (e.g., #my-id).

Box Model

width: Element width, which sets the width of the content area.

height: Element height, which sets the height of the content area.

padding: Inner space, which adds space between the content and the border of an element.

Text Properties

color: Text color, which sets the color of the text.

font-size: Text size, which sets the size of the text.

font-family: Font type, which sets the font family for the text.

Background Properties

background-color: Background color, which sets the background color of an element.

background-image: Background image, which sets an image as the background of an element.

Display Properties

display: inline: Inline element, which only takes up as much width as necessary and does not start on a new line.

display: inline-block: Inline-block element, which is formatted as an inline element, but you can set a width and height.

display: flex: Flex container, which enables flexbox layout.

display: grid: Grid container, which enables grid layout.

Positioning

position: relative: Relative position, where the element is positioned relative to its normal position.

position: absolute: Absolute position, where the element is positioned absolutely to its first positioned ancestor.

position: fixed: Fixed position, where the element is positioned fixed relative to the viewport.

Flexbox Properties

display: flex: Defines a flex container; inline or block depending on the given value.

flex-direction: Specifies the direction of the flexible items.

justify-content: Aligns the flexible container's items on the main axis.

Grid Properties

display: grid: Defines a grid container.

grid-template-rows: Specifies the number (and the height) of rows in a grid layout.

grid-gap: Specifies the gap between the grid lines.

Transform

transform: rotate(90deg): Rotates an element clockwise or counterclockwise.

transform: scale(1.2): Increases or decreases the size of an element.

Transitions

transition: background-color 0.3s ease: Creates smooth transitions.

transition: transform 0.3s ease: Specifies the CSS property the transition effect is for.
