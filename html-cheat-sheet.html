HTML Cheat Sheet

Basic Structure

<!DOCTYPE html>: Declares the document type as HTML5. It tells the browser that this is an HTML5 document.

<head>: Contains meta-information about the HTML page, such as title, character set.

<title>: Specifies a title for the HTML page, which is shown in the browser's title bar or tab.

<body>: Defines the document's body and contains all the content such as headings, paragraphs, images, and more.

Text Formatting

<h1> to <h6>: Defines HTML headings, with <h1> being the most important and <h6> the least.

<br>: Inserts a single line break.

<hr>: Defines a thematic break in an HTML page (horizontal rule).

<pre>: Defines preformatted text, which is displayed in a fixed-width font, and spaces and line breaks are preserved.

Links

<a href="url">link text</a>: Defines a hyperlink, which is used to link to other web pages or resources.

<a target="_blank" href="url">link text</a>: Opens the linked document in a new window or tab, using the target="_blank" attribute.

Images

<img src="url" alt="alt text">: Defines an image, where src specifies the URL of the image, and alt provides alternative text if the image cannot be displayed.

Lists

<ul>: Defines an unordered (bulleted) list.

<ol>: Defines an ordered (numbered) list.

<li>: Defines a list item, which is used inside <ul> or <ol> elements.

Tables

<table>: Defines a table.

<tr>: Defines a row in a table.

<th>: Defines a header cell in a table.

<td>: Defines a data cell in a table.

Forms

<form>: Defines an HTML form, which is used to collect user input.

<input>: Defines an input field, which can be of various types such as text, password, or email.

<button>: Defines a clickable button, which can be used to submit a form or perform other actions.

Semantic Elements

<header>: Defines a header for a document or section, typically containing the site title and navigation links.

<nav>: Defines a set of navigation links.

<footer>: Defines a footer for a document.

Meta Tags

<meta charset="UTF-8">: Defines the character set used for the HTML document, typically set to UTF-8.

<meta name="author" content="author">: Defines the author of the page.

<meta name="viewport" content="width=device-width, initial-scale=1.0">: Sets the viewport for responsive design.

Divs and Spans

<div>: Defines a division or a section in an HTML document..

<span>: Is an inline container used to mark up a part of a text..

Iframes

<iframe src="url"></iframe>: Used to embed another document within the current HTML document
